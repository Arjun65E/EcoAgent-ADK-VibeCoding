# EcoAgent: Multi-Agent Sustainability Analytics & Recommendation System

Built during the **Google × Kaggle 5-Day AI Agents: Intensive Vibe Coding Course (June 2026)**.

## Project Overview
EcoAgent is an automated urban planning and environmental intelligence application designed under the **Agents for Good** track. It utilizes a hierarchical multi-agent structure to fetch real-time climate data via custom Model Context Protocol (MCP) servers, perform analytical evaluation checks, and output strategic executive briefs to guide municipal sustainability decisions.

## Core Core Concepts Demonstrated
1. **Multi-Agent Orchestration (ADK):** Implements a primary Coordinator Agent managing sequential workflows across a distinct Data Analyzer Agent and a Report Synthesis Agent.
2. **Model Context Protocol (MCP):** Connects external contextual databases cleanly using a localized custom MCP simulator pattern.
3. **Session & State Management:** Features structural in-memory session mapping tracking system history across multiple agent interaction loops.
